# Printable Sudoku Generator

Minimal plain-JS Sudoku generator.

Features included:
- Full-board generator (backtracking)
- Puzzle creation with uniqueness checks
- Print-friendly CSS

To preview locally (simple static server):

PowerShell / command line:

```bash
# using Python 3 in the project folder
python -m http.server 8000
# or with npm http-server (if installed)
npx http-server -p 8000
